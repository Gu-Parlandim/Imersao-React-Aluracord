## 🚀 Instalando AluraCord

Para instalar o  AluraCord, siga estas etapas:


```
npm install
```
ou

```
yarn add [package]
```


## ☕ Usando AluraCord

Para usar AluraCord, siga estas etapas:

```
npm run dev
```
ou
```
yarn dev
```
